# Research Index

This folder contains all research underpinning Radical Moderate's product and strategic decisions.

## Structure

```
research/
  trends/       Core trends that create the market opportunity
  competitive/  Analysis of competitors and adjacent players
  concepts/     Research supporting specific product concepts
```

## How to Use

Each trend file follows this format:
- **Summary** — what the research shows
- **Strategic relevance** — why it matters for Radical Moderate
- **Key findings** — specific, citable data points
- **Sources** — full citations with URLs
- **Confidence** — how well-established or contested this finding is

All sources linked here should be directly accessible. Paywalled academic sources are noted; preprints or open-access versions are preferred where available.
